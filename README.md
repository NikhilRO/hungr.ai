# Dev Setup
*Make sure you have Python3*
1. `npm run venv`
2. `./env/Scripts/activate` to activate virtualenv
3. Run `npm run build`
4. Run `npm start`
5. Go to <a href="http://localhost:5000/" target="_blank">http://localhost:5000/</a>
6. `deactivate` to get out of virtualenv
